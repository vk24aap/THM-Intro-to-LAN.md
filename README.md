# THM-Intro-to-LAN.md
intro to LAN (Local Area Network)
Project Focus: Network Topologies, Switching, and Routing Fundamentals.

Objective: To analyze the design and flaws of private network architectures.

Key Learning: I mastered the differences between Star, Bus, and Ring topologies. I identified that the Star Topology is the enterprise standard due to its scalability and ease of troubleshooting via a central switch.

Technical Deep Dive: * Switches vs. Hubs: I explored how switches intelligently direct packets to specific ports using MAC addresses, reducing network congestion compared to hubs.

The Role of the Router: I practiced the concept of "Routing," which is the process of passing data between different networks (e.g., LAN to WAN).

SOC Value: Understanding LAN traffic is critical for detecting Lateral Movement. If an attacker compromises one device, an analyst must know the network topology to predict where they will move next.
